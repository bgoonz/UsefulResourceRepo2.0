Package Management

Context: frontend-dev-bookmarks / Workflow

A package manager or package management system is a collection of software tools that automates the process of installing, upgrading, configuring, and removing reusable libraries and components in a consistent manner.

+ Bower: Bower offers a generic, unopinionated solution to the problem of front-end package management, while exposing the package dependency model via an API that can be consumed by a more opinionated build stack. + Lerna: Lerna is a tool that optimizes the workflow around managing multi-package repositories with git and npm. + NPM: NPM makes it easy for JavaScript developers to share and reuse code, and it makes it easy to update the code that you're sharing.
