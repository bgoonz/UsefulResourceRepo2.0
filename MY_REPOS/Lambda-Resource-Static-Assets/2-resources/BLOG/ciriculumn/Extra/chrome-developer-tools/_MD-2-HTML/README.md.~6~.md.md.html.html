Self-Study Questions

Demonstrate your understanding of this week’s concepts by answering the following free-form questions.

Edit this document to include your answers after each question. Make sure to leave a blank line above and below your answer so it is clear and easy to read by your project manager

  1. If you were to describe semantic HTML to the next cohort of students, what would you say?

Most tags come in pairs and Prettier won’t be able to work if your HTML is broken, so make sure for every needing opening tag you have a closing tag. You don’t need to memorize them because you’ll most likely either use cheatsheets or will remember it over a period of time due to frequent use.

  1. Name two big differences between display: block; and display: inline;.

display: block; aligns items vertically while display: inline; aligns items horizontally.

  1. What are the 4 areas of the box model?

Margin -> Border -> Padding -> Content

  1. While using flexbox, what axis does the following property work on: align-items: center?

Cross axis, otherwise known as y-axis.

  1. Explain why git is valuable to a team of developers.

It gives an ability for multiple developers to work on a variety of features using the base at the same time while not breaking main code.

You are expected to be able to answer all these questions. Your responses contribute to your Sprint Challenge grade. Skipping this section will prevent you from passing this challenge.
