Cheat sheet: i3 Linux window manager

Learn shortcuts to become even more productive with i3.

i3_cheatsheet_coverimage.jpg

The i3 window manager is a lightweight tiling window manager for the Linux operating system.

The i3 window manager may seem hard to use at first because it relies on keyboard shortcuts for most operations. Use this cheat sheet to help you get around i3 when you're starting.
